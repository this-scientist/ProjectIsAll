Core Business Flow — Task Crowdfunding Lifecycle

5 Phases of State Transitions
1. Publish
Creator stakes tokens
Describes task & bounty
→
2. Crowdfund
Community funds
Activates on goal met
→
3. Build
Dev claims task
Submits deliverables
→
4. Multisig Verify
N/M signatures
Review & approve
→
5. Settlement
Pass → transfer
Fail → re-list
Timeout → auto-release
